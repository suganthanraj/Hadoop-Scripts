# Activity Description Start Script
1 Hadoop Single Node Cluster Installation Hadoop Installation on Single or Master Machine
2 Hadoop Multi Node Cluster Installation Hadoop Installation on multiple slave machine
3 Commissioning Nodes to the Cluster Enabling nodes to the Master Cluster in which nodes are already in the cluster
4 Decommissioning Nodes from the Cluster Disabling nodes to the Master Cluster in which nodes are already in the cluster
5 Adding Nodes to the Cluster Connecting New nodes to the Master Cluster in which nodes are not already in the cluster
6 Upgrade Hadoop Version Upgrade Hadoop Version in the Cluster
7 Refresh Nodes Refreshing All nodes available in the cluster
8 Hadoop Cluster Health Status Health Status of all the nodes in the cluster
9 Monitoring HDFS Diskspace Frequently monitoring the Hadoop File System Diskspace (HDFS) in the Cluster
10 Start Hadoop Daemon as a linux Services Set a Hadoop Daemon as a linux services
